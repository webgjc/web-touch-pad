# web-touch-pad
网页触控板

## 实现功能
- 箭头滑动
- 左键（单指点击）
- 右键（双指点击）
- 滚动（双指上下滑）

## 使用场景
- 讲ppt可在电脑上开启，将手机作为控制板点击切页
- 触控板失效的情况下，可以作为救急
- 远程遥控等等

## 运行

使用环境为Python3

安装必需的包，在requirements.txt中（也可以直接运行下看少什么就用pip装什么）

启动
> python main.py

## 启动后

启动后请按提示

> http://xxx.xxx.xxx.xxx:8000

请在局域网另一个设备进行访问

可将那个设备作为本设备的触控板

注意请将设备顺时针旋转90度使用

Mac的话需要开启安全与隐私 - 辅助控制 - 允许APP控制电脑

## 其他注意问题

- Werkzeug包必须是1.x.x的，2以上会有报错
